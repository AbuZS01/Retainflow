# Desktop Gate + Tap-to-Play Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show a QR-code landing page to desktop visitors; let mobile users tap an ayah marker to play audio from that ayah or play just that one ayah.

**Architecture:** Desktop gate is pure CSS (`pointer: fine` + `min-width` media query) with a static `#desktop-gate` div — no JS detection, no redirect, no server change except CSP. Tap-to-play adds a `data-ayah-idx` attribute to each `.ayah-marker` span, a positioned `#ayah-menu` popup, and a `singleAyahMode` flag on `audioState`.

**Tech Stack:** Vanilla HTML/CSS/JS frontend; Fastify backend (CSP header only); no new dependencies.

### Security note
The QR code is generated by `https://api.qrserver.com` — a free, widely-used, read-only image API. It encodes only the public app URL (no tokens, no user data). HTTPS is enforced by the API. We add it to `img-src` in the CSP so the browser allows the image; nothing else changes.

### How it works
Each `.ayah-marker` span gets a `data-ayah-idx` attribute (the 0-based index into `audioState.ayahs`). Tapping it calls `showAyahMenu(ayahIdx, markerEl)`, which positions `#ayah-menu` near the marker. Two buttons:
- **▶ Play from here** → `playFromIdx(ayahIdx)` (existing function)
- **▶ Play this ayah** → sets `audioState.singleAyahMode = true`, calls `playFromIdx(ayahIdx)`

In the `ended` handler of `playFromIdx`, if `singleAyahMode` is true, stop after the current ayah instead of advancing.

- [ ] **Step 5: Add `showAyahMenu` / `hideAyahMenu` functions in `app.js`**

Add these two functions immediately before `updateAudioUI` (search for `function updateAudioUI(`):

```javascript
function hideAyahMenu() {
  const menu = document.getElementById('ayah-menu');
  menu.classList.add('hidden');
  menu.removeAttribute('data-ayah-idx');
}

function showAyahMenu(ayahIdx, markerEl) {
  const menu = document.getElementById('ayah-menu');
  menu.setAttribute('data-ayah-idx', ayahIdx);

  // Position near the marker
  const rect    = markerEl.getBoundingClientRect();
  const menuW   = 180; // approx width from CSS
  const scrollY = window.scrollY || document.documentElement.scrollTop;

  let left = rect.left - menuW / 2 + rect.width / 2;
  left = Math.max(8, Math.min(left, window.innerWidth - menuW - 8));

  menu.style.top  = `${rect.bottom + scrollY + 6}px`;
  menu.style.left = `${left}px`;
  menu.classList.remove('hidden');
}
```
